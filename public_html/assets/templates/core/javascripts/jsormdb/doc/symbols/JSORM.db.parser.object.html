Class JSORM.db.parser.object

Parser to convert objects into objects, essentially making no translation. Because the database must have a parser for other conversions, e.g. JSON and XML, we need an object parser as well, especially when the channel talks to some other object generator.
Defined in: ObjectParser.js.
